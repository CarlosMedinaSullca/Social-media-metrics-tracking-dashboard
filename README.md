# Overview

This software had the purpose to implement avanced software development techiques like classes and recursion using Typescript.
This is a vanilla JavaScript dashboard that displays analized data from the Meta marketing API. The appliation uses native Array ES6 functions like filter, map, and reduce. It uses slint and prettier packages to improve code formating and coherence. Since social media APIs are notorious for rate-limiting or failing the application includes robust code by wrapping data fetching in try catch blocks. Styles were added dynamically using DOM.
